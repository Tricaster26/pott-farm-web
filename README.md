## How to run the server

First, run the development server:

```
npm install
```

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

If npm is not installed:

## Mac OS
```
brew install node
# and
brew install npm
```

## Windows

Download [node.js](https://nodejs.org/en/download/package-manager) and follow the steps listed

## Running te server

Then run :

```
npm install
```

followed by:

```
npm run dev
```
